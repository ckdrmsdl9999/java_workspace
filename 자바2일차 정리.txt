   Java Virtual Machine : 자바 코드 해석및 메모리를 관리
                                     하는 소프트웨어 

7.환경변수란 무엇이고 사용 목적은?
   OS에 등록하여 사용할 수 있는 변수를 의미하며 특히 path
   라는 변수에는 OS가 가동되고 있는 한 언제든 접근이 가능
   한 전역데이터를 등록할 수 있다!!
   어느 위치에서건 등록된 경로를 참조할 수 있다!!

8.자료형이란 무엇이고, 자바의 자료형의 종류 및 크기는?
   데이터의 종류를 의미하여 종류를 구분하는 이유는 메모리
   에 차지할 데이터의 용량을 결정하기 위함이다!!

   문자 : char(2)
   숫자 : byte(1) < short(2) < int(4) < long(8)
   논리값 : boolean(1)
